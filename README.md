# HalalInv – Application de conformité Sharia pour l’investissement  
## Version Portfolio

## Contexte
HalalInv est un projet de fin d’année réalisé **au sein de l’entreprise 6Solutions**, dans le cadre d’un projet académique encadré.  
L’objectif du projet est de concevoir et développer une plateforme web permettant d’analyser et de classifier des actions et des ETF selon leur conformité aux principes de la finance islamique (Sharia).

Ce projet s’inscrit dans un contexte professionnel réel, avec des exigences fonctionnelles et techniques proches d’un environnement de production.

---

## Objectifs du projet
- Analyser la conformité Sharia des entreprises cotées
- Classifier les actions et ETF selon des critères financiers précis
- Fournir une interface claire et intuitive pour la consultation des données
- Permettre la création et la gestion de watchlists personnalisées
- Centraliser des informations financières utiles à la prise de décision

---

## Fonctionnalités principales
- Authentification des utilisateurs
- Recherche d’entreprises et d’ETF
- Calcul de la conformité Sharia (Halal / Haram / Douteux)
- Consultation des ratios financiers
- Gestion de watchlists personnalisées
- Visualisation des informations financières et des performances

---

## Architecture
L’application repose sur une architecture web en trois couches :

- **Frontend** : Angular (Single Page Application)
- **Backend** : API REST développée avec Spring Boot
- **Base de données** : MySQL
- **Services externes** : API Financial Modeling Prep pour la récupération des données financières

Cette architecture permet une séparation claire des responsabilités et facilite la maintenabilité et l’évolution du système.

---

## Stack technique
- **Backend** : Java, Spring Boot
- **Frontend** : Angular
- **Base de données** : MySQL
- **API externe** : Financial Modeling Prep
- **Outils** : Maven, IntelliJ IDEA, Postman
